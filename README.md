## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/el3ttar3/DSAI-Course-Notebooks.git
   ```

2. **Navigate to a course folder**
   ```bash
   cd DSAI-Course-Notebooks/02-Deep-Learning
   ```

3. **Open in Google Colab** - Click the "Open in Colab" badge in any notebook

4. **Install dependencies** - Each notebook includes its required packages
